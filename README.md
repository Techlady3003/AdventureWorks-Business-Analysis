## Project Title

# AdventureWorks Business Analysis Dashboard


## Project Overview

This project analyzes AdventureWorks sales performance using Power BI. The dashboard provides insights into revenue trends, customer behavior, product performance, and regional sales distribution.


## Tools Used

- Power BI
- Power Query
- DAX
- Excel

- ## Dashboard Pages

### Executive Sales Overview
Provides high-level KPIs and revenue analysis.

### Product Performance Analysis
Identifies top-performing products and categories.

### Customer & Territory Analysis
Analyzes customer distribution and regional performance.

### Sales Trend Analysis
Tracks revenue and order trends over time.

## Key Insights

- Total Revenue: $109.85M
- Bikes generated approximately $95M in revenue.
- North America generated the highest revenue.
- Revenue peaked in 2013 at approximately $43.6M.
- Mountain-200 products were among the top revenue-generating products.

- ## Recommendations

- Invest more in high-performing bike categories.
- Improve sales strategies in lower-performing regions.
- Explore opportunities to grow accessory and clothing sales.
- Use customer insights to improve targeted marketing campaigns.
